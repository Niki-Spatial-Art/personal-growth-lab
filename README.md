# Personal Growth Lab

个人成长方向的公开框架仓库。

这个仓库用于整理健康、美学、训练、习惯、状态复盘和长期自我管理系统。它只保留公开友好的方法论、模板、资料索引和可复用流程，不放敏感个人信息。

## Reference Scope

当前框架可以先围绕一个轻量体型参考范围维护：

- Height: about 173 cm
- Weight: about 55 kg
- Focus: body composition, posture, skin barrier, beauty routines, sustainable habits

这个范围只用于筛选资料、训练计划和生活方式模板，不作为医疗建议或个人隐私档案。

## What Goes Here

- 健康、美容、护肤、体态管理资料
- 训练和营养计划框架
- 习惯系统和状态复盘模板
- 值得收藏或 fork 的个人管理 GitHub 项目
- 可公开复用的方法论和 checklist

## What Does Not Go Here

- 身份证件、地址、手机号、账户信息
- 详细体检报告、医疗诊断、病历
- 私人照片、聊天记录、具体行程
- 任何不适合公开传播的个人状态记录

## Structure

- `repos.md`: 个人成长相关 GitHub 仓库索引
- `playbooks/`: 训练、复盘、计划流程
- `templates/`: 状态复盘、计划、仓库登记模板
- `archive/`: 历史记录和旧流程

## Maintenance Rule

后续定期补充资料时，优先加入：

1. 有明确出处的健康、美容、训练资料。
2. 可复用的习惯、计划、复盘工具。
3. 不涉及隐私的公开模板。
4. 适合 173 cm / 55 kg 轻量体型参考范围的训练和体态管理框架。

## Privacy Note

This repository is public. Keep it framework-based and reusable. Do not commit sensitive health, identity, finance, photos, private life details, or account information.
