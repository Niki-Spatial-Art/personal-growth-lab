# Personal Growth Lab

个人成长方向的私有维护仓库。

这个仓库用于整理健康、美学、训练、习惯、状态复盘和长期自我管理系统。它默认先保持私有，公开内容只保留方法论和模板，不放敏感个人数据。

## What Goes Here

- 健康、美容、护肤、体态管理
- 训练和营养计划
- 习惯系统
- 周度状态复盘
- 值得收藏或 fork 的个人管理 GitHub 项目

## Structure

- `repos.md`: 个人成长相关 GitHub 仓库索引
- `playbooks/`: 训练、复盘、计划流程
- `templates/`: 状态复盘、计划、仓库登记模板
- `archive/`: 历史记录和旧流程

## Privacy Note

Do not commit sensitive health, identity, finance, or private life details into this repository unless you intentionally keep it private.
